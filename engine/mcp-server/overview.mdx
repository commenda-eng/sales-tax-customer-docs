## Overview

The Commenda MCP server exposes Commenda APIs as tools for AI assistants that support the [Model Context Protocol (MCP)](https://modelcontextprotocol.io/). Once connected, your AI assistant can manage business entities, compliance items, sales tax filings, transactions, customers, products, and more — all through natural language.

<Note>
  MCP is an open standard that lets AI assistants call external tools. The Commenda MCP server implements this protocol so tools like Claude Desktop, Cursor, and other MCP-compatible clients can interact with your Commenda data.
</Note>

## When to use this

Use the MCP server when you want to:

- Discover which organizations and entities you have access to
- Query and update business entity data from an AI assistant
- Manage compliance calendars and filings conversationally
- Look up nexus information and registration thresholds
- Manage sales tax registrations, filings, and onboarding
- Create or update customers, products, and exemption certificates
- Calculate tax rates without leaving your AI workflow

## Available tool categories

The MCP server provides **95+ tools** across the following categories:

| Category | Examples |
| --- | --- |
| **Organizations** | List all organizations (companies, firms) you have access to |
| **Business entities** | List, get, and update business entities |
| **Locations** | Create, update, list, get, and delete entity locations |
| **Government identifiers** | Create, update, list, and get identifiers (EIN, PAN, TRN, etc.) |
| **Registrations** | Manage tax and corporate registrations; get supported types |
| **Documents** | List, create, update, and get supported document types |
| **Persons** | Add or remove key persons (directors, officers, shareholders) |
| **Credentials** | Link credentials to entities or registrations |
| **Compliance** | Create, update, delete, and bulk-update compliance items; generate calendars; manage extensions; get accountant inputs |
| **Sales tax filings** | List, update, approve, submit, and delete filings; generate and preview filing calendars; manage assignees |
| **Transactions** | List, get, commit, lock, void, and bulk-delete transactions; get aggregates and line items |
| **Nexus** | Get nexus data, recalculate tax estimates, regenerate, clear, and resync |
| **Sales tax registrations** | Create, update, delete, list, and validate registrations |
| **Customers** | CRUD operations and exemption lookups |
| **Products** | CRUD operations and tax code lookups |
| **Exemption certificates** | List, create, update, and delete certificates |
| **Tax calculations** | List calculations, get details, and calculate tax rates |
| **Sales tax onboarding** | Get onboarding status including products, transactions, and registrations |

## Deployment modes

The MCP server supports two deployment modes:

- **Stdio** — for local use with Claude Desktop, Cursor, and similar tools. The AI client launches the server process directly.
- **HTTP** — runs as a sidecar service on port 3001 for remote or shared access.
